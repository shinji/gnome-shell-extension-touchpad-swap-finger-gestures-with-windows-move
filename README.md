# Touchpad swap finger gestures with windows move

GNOME Extension that allows change workspace with four fingers and windows move with three fingers swipe.

Based on:

[Three Finger Window Move](https://github.com/do-sch/gnome-shell-touchpad-window-move)

[Four Finger Window Move](https://github.com/mechtifs/gnome-shell-touchpad-window-move)

[Swap Finger Gestures (3 to 4)](https://github.com/icedman/swap-finger-gestures-3-4)
